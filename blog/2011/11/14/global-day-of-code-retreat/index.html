Global Day of Code Retreat

Nov 14th, 2011 | Comments

We would like to invite everyone to a Code Retreat, which will take place in Cracow on the 3rd of December 2011 as a part of Global Day of Code Retreat.

You can find all of the details on the event site: coderetreat.sckrk.com

Event registration will open 14th November, 7pm CET. Admission is free, but the seating is limited, so be sure to sign up fast if you are interested in coming.

The Code Retreat would not be possible without the help of our generous sponsors: ABB, Applicake Future Simple, Kanbanery and Lunar Logic Polska.
